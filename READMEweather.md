# Консольное приложение для получения погоды

Простое CLI-приложение для получения информации о погоде по названию города или координатам.

## Функциональность

- Получение погоды по названию города
- Получение погоды по географическим координатам (широта и долгота)
- Кэширование полученных данных для уменьшения количества запросов к API
- Обработка ошибок и повторные попытки запросов при сетевых сбоях

## Требования

- Python 3.7+
- Библиотеки: requests, python-dotenv

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Настройка

Для работы приложения необходим API-ключ от OpenWeatherMap:

1. Зарегистрируйтесь на сайте [OpenWeatherMap](https://openweathermap.org/api)
2. Получите бесплатный API-ключ
3. Создайте файл `.env` в корневой директории проекта
4. Добавьте в него строку:
```
API_KEY=ваш_ключ_здесь
```

## Использование

Запустите приложение:

```bash
python weather_cli.py
```

В главном меню вы можете выбрать:
- 1 — получить погоду по названию города
- 2 — получить погоду по координатам
- 0 — выход из программы

## Архитектура

- `weather_cli.py` — основной интерфейс командной строки
- `weather_app.py` — логика получения погоды и геокодирования
- `weather_cache.py` — система кэширования данных
- `weather_cache.json` — файл для хранения закэшированных данных

## Особенности

- Приложение автоматически кэширует полученные данные на 3 часа
- При проблемах с сетью или ошибке API приложение использует кэшированные данные
- Реализован механизм повторных попыток запросов при временных ошибках
- Поддержка русского языка в описании погоды
